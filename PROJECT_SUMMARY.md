# Project Summary: TelcoGPT

## Project Title
TelcoGPT – 5G Network Operations Copilot

## Objective
To build a Hugging Face-powered RAG chatbot that supports telecom NOC engineers by answering LTE/5G troubleshooting questions from technical documents.

## Main Use Cases
- Explain RRC connection failure
- Troubleshoot packet loss
- Support eNodeB/gNodeB incident handling
- Explain handover failure
- Generate incident summaries

## Main Technologies
- Python
- Hugging Face
- LangChain
- FAISS
- Streamlit
- Sentence Transformers
- RAG

## Career Value
This project is suitable for AI Engineer, Generative AI Engineer, Data Scientist, MLOps Engineer, and Telecom AI roles.
